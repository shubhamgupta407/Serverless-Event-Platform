# 🚀 Serverless Event Platform

A scalable, real-time event management web application built using modern serverless architecture.
Users can create, manage, and explore events seamlessly with instant updates and a responsive UI.

---

## 🌐 Live Demo

[eventplatform1.onrender.com](https://eventplatform1.onrender.com)

> ⚠️ App may take a few seconds to load due to free hosting.

---

## ✨ Features

* 🔐 Secure user authentication (Firebase Auth)
* 📅 Create, update, and manage events
* ⚡ Real-time database updates (Firestore)
* 🎯 Role-based dashboards (Admin/User)
* 📱 Fully responsive UI
* 🎟️ Event registration system

---

## 🧠 Tech Stack

### Frontend

* React.js

### Backend / Services

* Firebase Authentication

## ⚙️ Installation & Setup

```bash
# Clone the repository
git clone https://github.com/shubhamgupta407/Serverless-Event-Platform.git

# Navigate to project
cd Serverless-Event-Platform

# Install dependencies
npm install

# Run the app
npm run dev
```
